# 慶應シラバス認証済みメガ・スクレイパー

慶應義塾大学のシラバス（gslbs.keio.jp）から、認証が必要な項目を含めて大量（2万件規模）に取得し、CSV/JSON 形式で出力するためのツール群です。

## 特徴
- **認証対応**: 多要素認証 (MFA) が必要な `keio.jp` ログインに対応。
- **大量取得**: 2万件の規模でも安定して動作し、中断・再開（レジューム）が可能。
- **画面なし実行**: ログイン時以外はブラウザを非表示（ヘッドレス）で実行可能。
- **動的項目対応**: 途中で新しい項目（列）が増えても自動で検出し、CSV に反映します。

## ファイル構成
- `start_chrome_debug.sh`: 手動ログイン用の Chrome をデバッグモードで起動。
- `save_auth.py`: ログイン済みのセッションを `auth.json` に保存。
- `mega_scraper.py`: メインの取得スクリプト（JSONL 形式で逐次保存）。
- `jsonl_to_csv.py`: 取得した JSONL データを CSV および一括 JSON に変換。
- `peek_syllabus.py`: 特定の ID の内容をターミナルでクイック確認するツール。

## 基本的な使いかた (VS Code 推奨)

VS Code の「実行とデバッグ」パネルから、以下の番号順に操作するだけで完結します。

### 1. ログインと認証情報の準備
1.  **`[1] ブラウザ起動`** を実行して Chrome を立ち上げる。
2.  ブラウザで `keio.jp` にログインし、MFA を完了させる。
3.  **`[2] 認証情報を保存`** を実行して `auth.json` を作成。
4.  立ち上げた Chrome ウィンドウは閉じて OK。

### 2. データの取得
-   **`【本番】シラバス取得 (2万件)`** を実行。
-   途中で止めたくなったら `Ctrl+C` で中断。再度実行すれば続きから再開されます。

### 3. データのエクスポート
-   **`【出力】CSV/JSON 生成`** を実行。
-   `syllabus_2026.csv` および `syllabus_2026.json` が生成されます。

## コマンドラインでの直接実行
VS Code を使わない場合は、以下のコマンドを順に実行してください。

```bash
# 1. ログイン情報を保存
python3 save_auth.py

# 2. 取得開始 (例: 1000件分)
python3 mega_scraper.py --headless --batch 1000

# 3. CSV/JSON 出力
python3 jsonl_to_csv.py
```

## 注意事項
- ログインセッションが切れた場合は、再度「1. ログインと認証情報の準備」からやり直してください。
- 取得速度は `time.sleep(0.5)`（秒）で調整されています。
