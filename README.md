# 慶應シラバス認証済みメガ・スクレイパー

慶應義塾大学のシラバス（gslbs.keio.jp）から、認証が必要な項目を含めて大量（2万件規模）に取得し、CSV/JSON 形式で出力するためのツール群です。

## 特徴
- **爆速取得 (並列対応)**: 昇順 (1→) と 降順 (99999→) の同時実行により、取得時間を最大で半分に短縮。
- **認証対応**: 多要素認証 (MFA) が必要な `keio.jp` ログインに対応し、ヘッドレス（画面なし）で実行可能。
- **レジューム機能**: 途中で中断しても、進捗ファイルから自動で続きから再開。
- **自動クエリ・デデュープ**: 複数プロセスで重複して取得しても、書き出し時に自動で重複を除去して整列。

## ファイル構成
- `start_chrome_debug.sh`: 手動ログイン用の Chrome をデバッグモードで起動。
- `save_auth.py`: ログイン済みのセッションを `auth.json` に保存。
- `mega_scraper.py`: メインの取得スクリプト。
- `jsonl_to_csv.py`: JSONL データを CSV/JSON に変換し、重複を除去。
- `progress_<year>_<direction>.json`: 年度と方向ごとの進捗管理ファイル。

## 基本的な使いかた (VS Code 推奨)

VS Code の「実行とデバッグ」パネル（F5キー）から、以下の順に操作します。

### 1. ログインと認証の準備
1.  **`[1] ブラウザ起動`** を実行してログインと MFA を完了させる。
2.  **`[2] 認証情報を保存`** を実行して `auth.json` を作成。ブラウザは閉じて OK。

### 2. 並列取得で時間を節約する（推奨）
1.  **`【取得】昇順 (1 →)`** を選んで実行。上部に表示されるプロンプトに年度（例: `2026`）を入力。
2.  **もう一度別のコンソールで `【取得】降順 (99999 →)`** を選んで実行。同じ年度を入力。
3.  これで両端から同時に取得が始まります。

### 3. データのエクスポート
-   **`【出力】CSV/JSON 生成`** を実行。
-   `syllabus_<年度>.csv` および `syllabus_<年度>.json` が生成されます。
-   **重複排除**: 昇順と降順が真ん中でぶつかって重複しても、出力時に自動で 1 つにまとめられます。

## トラブルシューティング

### ログインが切れた場合
『認証エラー』が出たら、再度「1. ログインと認証の準備」を行ってください。

### 途中で止まった場合
再度実行すれば、自動的に前回の続きから再開されます。

### 注意事項
- 大量取得時は PC のスリープ設定をオフにし、電源を接続してください。
- 取得速度は `time.sleep(0.5)` で調整されています。
